# Personal Portfolio Website

## Overview
A responsive personal portfolio website designed to showcase technical skills, projects, and interests in technology and cybersecurity.

## Features
- Responsive website layout
- Project showcase section
- Skills and contact sections
- Interactive navigation
- Clean UI/UX design

## Tools Used
- HTML
- CSS
- JavaScript

## Files
- `index.html` – Main webpage structure
- `style.css` – Website styling
- `script.js` – Interactive functionality

## What I Learned
- Frontend web development fundamentals
- Responsive web design
- UI/UX design principles
- Website structuring and styling
